# Gradium 

> This is a Freelance platform for College Students, So that they can Earn while learning New things ! 
---
> This Also helps Individuals/Clients to get their work done at Low-Cost !

> Check the `Hackathon-Files` Folder to see the Videos

## Frontend Tech Stack (Aryan Sinha): 
> Github Profile Link: [Aryan Sinha](https://github.com/Aryansinha2811)
1. React.js with TypeScript 
2. Tailwind CSS for modern UI

## Backend Tech Stack (Ansal Pandey):
> Github Profile Link: [Ansal Pandey](https://github.com/Ansalpandey)
1. Node.js with Express.js
2. MongoDB for database
3. JWT for authentication
4. RESTful API architecture
